## Summary

Installs configuration, neovim and my development tools

## Quickstart

```shell
mkdir -p ~/workspaces/
cd workspaces/
git clone https://github.com/giancarlo-misasi/dotfiles
cd dotfiles
./install.sh link_dotfiles
./install.sh install_system
./install.sh install_ohmyzsh
./install.sh install_mise
./install.sh install_tools_and_languages
```
